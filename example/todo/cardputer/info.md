# Cardputer Pin Map

## SPM1423 MIC

| SPM1423 | ESP32-S3FN8 |
| :--- | :--- |
| DAT | G46 |
| CLK | G43 |

## microSD Socket

| microSD Socket | ESP32-S3FN8 |
| :--- | :--- |
| CS | G12 |
| MOSI | G14 |
| CLK | G40 |
| MISO | G39 |

| DAT | G35 |

## Keyboard & Battery Detect

| Function | Pin Name | ESP32-S3FN8 |
| :--- | :--- | :--- |
| Battery Detect | ADC | G10 |
| Keyboard | Row Inputs (Y7~Y0) | G13, G15, G3, G4, G5, G6, G7 |
| Keyboard | Col Select A2 (74HC138) | G11 |
| Keyboard | Col Select A1 (74HC138) | G9 |
| Keyboard | Col Select A0 (74HC138) | G8 |
